# Agent Lookup Source Repo

This repository contains an integrity-first protocol for agent operations and governance.

See `protocol/protocol.json` for the operational protocol and integrity protocol templates.

Quick start

- Review `protocol/protocol.json`
- Run JSON validation locally:

```bash
python -m json.tool protocol/protocol.json
```
